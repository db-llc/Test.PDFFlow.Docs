Creating Document

Brief

The article describes how to create a PDF document.

Details

Class DocumentBuilder lets you to create a new PDF document, add and configure one or more sections of the document, and build a document.

To create a PDF document:

  • Create new document builder using DocumentBuilder's New method.

  • Add Sections.

    New method returns DocumentBuilder, so you can then call its methods AddSection, AddSection, AddSection, AddDefaultSection, AddDefaultSection to add Section and document content.

    You can create several Sections in one document. Each section may consist of several pages, they will be generated automatically depending on the content.

    For instructions on how to create and configure a Section, see Configuring section.

  • At the end of adding sections and content, you have to call Build(fileName) or Build(outputStream) methods to generate the document.
