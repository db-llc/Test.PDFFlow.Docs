Namespace Gehtsoft.PDFFlow.Models.Document

Brief

Namespace stores classes of main document parts (e.g. section, layout, section content...)

Details

Classes

Document

Represents document. Stores Sections collection and Dispose method

Layout

Holds layout pages, used for repeating areas. Only one layout available per Section.

LayoutArea

Area which can be added to LayoutPage to show repeating headers or footers

LayoutPage

Represents a page holding repeating areas (headers or footers)

OutlineElement

Stores outline element properties

Page

Represents page object (numeration style, orientation, size, margins...)

Section

Class for work with a section of a document

SectionContent

Stores content of the section as collection of content items
