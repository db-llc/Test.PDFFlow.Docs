Configuring Section

Brief

The article describes how to add and configure a Section.

Details

A Section is a part of a document holding Content and Layout.

You can create several Sections in one document. Each section may consist of several pages that will be generated automatically depending on the content.

To create a Section of the document:

  • Call AddSection, AddSection or AddSection methods of the DocumentBuilder class.

  • You can also quickly add section with default settings (margins 20, paper size A4, portrait orientation and Arabic numeration style) by calling AddDefaultSection or AddDefaultSection(Action configure), and then change certain parameters as needed.

    First method returns a Section, so you can call section's extension methods on the fly in one line, e.g. builder.AddDefaultSection().SetMargins(35).SetNumerationStyle(NumerationStyle.UpperLatin).SetOrientation(PageOrientation.Landscape).SetSize(PaperSize.A2).AddParagraph("Paragraph text")...

    Using second method with a configurer, you can set common Section properties and add document content in the action.

  • Set Section's settings (e.g. Margins), stored in Section.Page property.

    Full list of available settings is described in Page class.

    Changing settings is available with Section's extension methods from SectionBuilder (e.g. SetMargins, SetSize, SetOrientation).

    Section's numeration style can be changed by calling it's SetNumerationStyle method. Then you can print page number using AddPageNumber Paragraph's extension method and see numeration in the desirable format (Arabic, UpperRoman, LowerRoman, UpperLatin, LowerLatin, UpperCyrillic, LowerCyrillic).

  • Add content to the Section.

For a complete list of extension methods of a Section, see SectionBuilder.

Adding content to the Section is described in the following topics: Add Paragraph, Adding content to the Paragraph, Configuring Font, Add Image, Create Multilevel List, Add Table, Add Line, Add StackFlow, Create Outline.

Each section may have its own repeating layout areas, both headers and footers. For instruction on how to configure layout areas, see Configuring Repeating Area.

AddDefaultSection() with configurer: [show]

AddDefaultSection() with configurer: [hide]

       Section s = builder.AddDefaultSection(s =>
       {
       s.SetOrientation(() => PageOrientation.Landscape);
       s.SetNumerationStyle(NumerationStyle.UpperRoman);
       s.SetMargins(35);
       s.SetSize(PaperSize.A5);
       s.AddParagraph(p =>
       {
           p.TextOverflowAction = TextOverflowAction.Ellipsis;
           p.AddText("Lorem ipsum");
           p.SetFont(documentFont);
       });
       });
