Configuring Paragraph

Brief

The article describes how to add and configure a Paragraph.

Details

A Paragraph is one of the main content parts of a document used for generating text of different styles.

Section can have as many Paragraphs as you need.


To add a Paragraph to a Section, you must call one of the methods:

AddParagraph (adds empty Paragraph to a Section and returns Paragraph; you can add content and set font later using Paragraph's extension methods),

AddParagraph (adds Paragraph with text element to a Section; default Font is used, you can change it later using Paragraph's extension methods),

AddParagraph (adds Paragraph to the Section and helps you to configure it using action)

from ParagraphBuilder.


You can directly set public paragraph properties using a configurer of AddParagraph, or set paragraph properties and add paragraph content using extension methods of Paragraph implemented in ParagraphBuilder, such as SetFirstLineIndent, SetLineSpacing, SetJustifyAlignment, SetTextDirection, SetTextOverflowAction, SetWidowMinimum, SetOrphanMinimum.

As Paragraph is derived from ContentItem, you can set various Paragraph's properties using methods of ContentItemBuilder, such as SetMargins, SetAlignment, SetBackColor, etc...

You can set Paragraph's Border of different width, stroke style and color. SetBorder helps to configure content item borders using action. SetBorderStyle sets ContentItem's Border stroke style (None, Solid, Dashed, Dotted or Double). SetBorderColor changes color and SetBorderWidth changes widths of the Border.


Adding content to the Paragraph is described in Adding content to the Paragraph article.

Setting Font of the Paragraph or TextElement is described in Configuring Font article.

For instructions on how to create a MultiLevel List from the paragraph, see Creating Multilevel List.

For instructions on how to create an Outline, see Creating Outline.

   // Create new section:
   Section s = builder.AddDefaultSection();
 
   // Create empty paragraph:
   Paragraph par = s.AddParagraph();
 
   // Set properties:
   par.SetBackColor(Color.Blue).SetBorderColor(Color.Red).SetBorderWidth(3f).SetBorderStyle(Stroke.Dotted).SetFirstLineIndent(10f).SetFontColor(Color.Yellow).AddText("Lorem ipsum 1");
 
   // Create paragraph with text:
   s.AddParagraph("Lorem ipsum 2").SetMargins(20).SetAlignment(HorizontalAlignment.Right);
 
   // Configuring Paragraph:
 
   s.AddParagraph(p =>
   {
   	p.Alignment = HorizontalAlignment.Center;
   	p.BackgroundColor = Color.Gray;
   	p.SetBorder(b =>
   	{
   		b.SetWidth(3);
   		b.SetStyle(Stroke.Solid);
   		b.SetColor(Color.FromRgba(0.0, 1.0, 0.0));
   	});
   	p.IsOutline = true;
   	p.OutlineLevel = 0;
   	p.SetFont(new Font {Name = "Courier", Size = 16f, Color = Color.Green});
   	p.AddText("Lorem ipsum 3");
   });
