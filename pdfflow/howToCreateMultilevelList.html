Creating Multilevel List

Brief

The article describes how to create a Multilevel List.

Details

Each Section may have as many Multilevel Lists as you need.

By setting corresponding Paragraph properties you can make it to be a part of a Multilevel List.

First, you must create a new ListElement and set it to Paragraph's List property:

  • Call SetListNumbered to make a Paragraph an item of MultiLevel list with number numeration (Arabic, UpperLatin, LowerRoman...). Set numeration style, level and indent:

    p.SetListNumbered(NumerationStyle.UpperRoman, 0, 15);

  • Call SetListBulleted method to make a Paragraph an item of MultiLevel list with bullet or dash in the beginning. Set bullet style (Bullet or Dash), level and indent:

    p.SetListBulleted(ListBullet.Dash, 1, 10);

  • Or assign it directly: p.List = new ListElement(ListStyle.Bullets, ListBullet.Bullet, null, 0, font.Size.Value * 2F);

Note that Paragraph.List.Style must be set for the paragraph to be a part of the Multilevel List.

Once paragraph with List is null or List.Style is null met in the Section, the multilevel list is considered completed.

When later another paragraph with not null List.Style is found, it will be a new Multilevel List, and it's automatic numeration will be started from the beginning.

The ListStyle enumeration can be either Bullets or Numbered.

If the Bullets style is set, you must also set the List.Bullet property, otherwise, the exception will be thrown. This property is indicating the appearance of bullet (black round or dash).

If the Numbered style is set, you must also set the List.NumerationStyle property, otherwise, the exception will be thrown. NumerationStyle can be arabic, lower/upper roman, lower/upper latin, lower/upper cyrillic.

In case of the Numbered style of the list, the numeration on every level will be done automatically.

The List.Level property will help you with this (please, see the screenshot in the end of this article).

By setting List.LevelLeftIndent you can control how levels are shifted from the upper levels.

For a complete list of Paragraph's List properties, see ListElement.

MultiLevel List: [hide]

        var font = Font.Courier(20).SetColor(Color.FromRgba(201.0 / 255, 107.0 / 255, 20.0 / 255)).SetBold();
        float leftIndent = font.Size.Value * 2F;
        s.AddParagraph("Level = 0. Item 1.").SetAlignment(HorizontalAlignment.Left).SetFont(font).SetListBulleted(ListBullet.Bullet, 0, leftIndent);
        s.AddParagraph("Level = 0. Item 2.").SetAlignment(HorizontalAlignment.Left).SetFont(font).SetListBulleted(ListBullet.Bullet, 0, leftIndent);
        s.AddParagraph("Level = 1. Item 1.").SetFont(font).SetFontSize(16).SetListNumbered(NumerationStyle.Arabic, 1, leftIndent);
        s.AddParagraph("Level = 2. Item 1.").SetFont(font).SetFontSize(12).SetListNumbered(NumerationStyle.UpperLatin, 2, leftIndent);
        s.AddParagraph("Level = 2. Item 2.").SetFont(font).SetFontSize(12).SetListNumbered(NumerationStyle.UpperLatin, 2, leftIndent);
        s.AddParagraph("Level = 1. Item 2.").SetAlignment(HorizontalAlignment.Left).SetFont(font).SetFontSize(16).SetListNumbered(NumerationStyle.Arabic, 1, leftIndent);
        s.AddParagraph("Level = 0. Item 3.").SetAlignment(HorizontalAlignment.Left).SetFont(font).SetListBulleted(ListBullet.Bullet, 0, leftIndent);

        var font = new Font
        {
            Name = FontNames.Courier,
            Size = 20f,
            Color = Color.FromRgba(201.0 / 255, 107.0 / 255, 20.0 / 255),
            Bold = true
        };
        var font1 = new Font
        {
            Name = FontNames.Courier,
            Size = 16f,
            Color = Color.FromRgba(201.0 / 255, 107.0 / 255, 20.0 / 255),
            Bold = true
        };
        var font2 = new Font
        {
            Name = FontNames.Courier,
            Size = 12f,
            Color = Color.FromRgba(201.0 / 255, 107.0 / 255, 20.0 / 255),
            Bold = true
        };
        float leftIndent = font.Size.Value * 2F;
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font);
                element.AddContent("Level = 0. Item 1.");
            });
            p.List = new ListElement();
            p.List.Style = ListStyle.Bullets;
            p.List.Level = 0;
            p.List.LevelLeftIndent = leftIndent;
            p.List.Bullet = ListBullet.Bullet;
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font);
                element.AddContent("Level = 0. Item 2.");
            });
            p.List = new ListElement(ListStyle.Bullets, ListBullet.Bullet, null, 0, leftIndent);
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font1);
                element.AddContent("Level = 1. Item 1.");
            });
            p.List = new ListElement(ListStyle.Numbered, null, NumerationStyle.Arabic, 1, leftIndent);
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font2);
                element.AddContent("Level = 2. Item 1.");
            });
            p.List = new ListElement(ListStyle.Numbered, null, NumerationStyle.UpperLatin, 2, leftIndent);
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font2);
                element.AddContent("Level = 2. Item 2.");
            });
            p.List = new ListElement(ListStyle.Numbered, null, NumerationStyle.UpperLatin, 2, leftIndent);
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font1);
                element.AddContent("Level = 1. Item 2.");
            });
            p.List = new ListElement(ListStyle.Numbered, null, NumerationStyle.Arabic, 1, leftIndent);
        });
        s.AddParagraph(p =>
        {
            p.Alignment = HorizontalAlignment.Left;
            p.AddText((element) =>
            {
                element.SetFont(font);
                element.AddContent("Level = 0. Item 3.");
            });
            p.List = new ListElement(ListStyle.Bullets, ListBullet.Bullet, null, 0, leftIndent);
