Configuring Font

Brief

The article describes how configure Fonts.

Details

Library allows to use different styles and Fonts in Paragraph.

Font can be of different size and color. It can have backgound color, be underlined or crossed out by the line of different stroke style and color. Font can be Bold, Italic or Oblique.

Please refer to Font for the full list of Font's properties.

You can use different Fonts either for the whole Paragraph or for TextElement.

For the Paragraph Font please use methods from ParagraphBuilder, such as SetFont, SetFontSize, SetFontColor, SetBold, SetStrikethrough...

For TextElement's Font please use methods from FormattedTextElementBuilder, such as SetFont, SetFontName, SetUnderline, SetUnderlineColor...

You can also directly change some Font properties using Font's extension methods from ParagraphBuilder, such as SetBold, SetOblique, SetColor, SetItalic.

Some of the Fonts are included into the library. Please, use FontNames (e.g. FontNames.Courier) or string (e.g. "Times-Italic").

Extra fonts can be imported from ttf files using LoadFontFromTtfFile method (please see example in the end of this article).

You can add text to paragraph without specifying any Font, in this case default Font will be used (Courier, size 14, color Black).


A few tips:

To get text of different styles successively, not separated with a new line, call AddText several times for one paragraph and set different font for each text element individually. Please refer various examples below for more information.

Use parentFont.Merge(otherFont) to apply settings of otherFont to parentFont.

Please see some examples below.

More examples using Fonts can be found in Quick Start, Configuring Paragraph, Adding content to the Paragraph articles.

   var p = new Paragraph();
   p.AddText((element) =>
   {
       element.SetFont(new Font
       {
           Name = "Courier",
           Size = 14f,
           Color = Color.Black
       });
       element.AddContent("Black font ");
   });
 
   p.AddText((element) =>
   {
       element.SetFont(new Font
       {
           Name = "Courier-Bold",
           Size = 16f,
           Color = Color.Red,
       });
       element.AddContent("large red underlined font ");
       element.SetUnderline(Stroke.Solid);
       element.SetUnderlineColor(Color.FromRgba(201.0 / 255, 107.0 / 255, 20.0 / 255));
   });
 
   p.AddText((element) =>
   {
       element.SetFont(new Font
       {
           Name = "Courier",
           Size = 10f,
           Color = Color.Blue
       });
       element.AddContent("blue small font.");
   });
   section.AddParagraph(p);

External Fonts from ttf files: [hide]

   var exofontsDir = Path.Combine(ResourceDirectory, "fonts", "exo");
   if (Directory.Exists(exofontsDir))
   {
   	var exoRegular = Path.Combine(exofontsDir, "Exo-Regular.ttf");
   	var exoRegularItalic = Path.Combine(exofontsDir, "Exo-Italic.ttf");
   	var exoRegularBold = Path.Combine(exofontsDir, "Exo-Bold.ttf");
   	var exoRegularBoldItalic = Path.Combine(exofontsDir, "Exo-BoldItalic.ttf");

   	_nativeDocument.LoadFontFromTtfFile(exoRegular, true);
   	_nativeDocument.LoadFontFromTtfFile(exoRegularItalic, true);
   	_nativeDocument.LoadFontFromTtfFile(exoRegularBold, true);
   	_nativeDocument.LoadFontFromTtfFile(exoRegularBoldItalic, true);
   }
   else
   {
   	_nativeDocument.AddFontOverride(FontNames.Exo, FontNames.Helvetica);
   }
