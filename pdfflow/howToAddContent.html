Adding content to Paragraph

Brief

The article describes how to add and configure content of a Paragraph.

Details

In Configuring Paragraph article is described how to add Paragraph to the Section and set it's properties.

There are few ways to add text content to the Paragraph:

  • Add empty paragraph using AddParagraph(this Section section) and then call one of the following:

    • AddText(this Paragraph paragraph, string text) (adds string to the Paragraph's content)

      AddText(this Paragraph paragraph, string[] text) (adds array of strings to the Paragraph's content)

      AddText(this Paragraph paragraph, Action<TextElement> elementConfig) (adds TextElement to the Paragraph's content using configurer action):

      p.AddText((element) =>
      {
      var font = Font.Courier(14);
      element.SetFont(font);
      element.AddContent("Some content");

    • AddTextToParagraph(this Paragraph paragraph, string text).

      This method adds string to Paragraph's content and returns Paragraph. This allows you to call AddTextToParagraph again in the same line, or call one of the AddText methods returning TextElement.

      s.AddParagraph() .AddTextToParagraph("Text New 1, ").AddTextToParagraph("Text New 2, ").AddTextToParagraph("Text New 3")
      .SetFont(documentFont).SetFontName(FontNames.Courier).SetStrikethrough(Stroke.Solid).SetFontSize(10)
      .AddText("Text New 3").SetFontName(FontNames.Times).SetFontSize(8).SetStrikethrough(Stroke.None);

  • Add paragraph with text using AddParagraph(this Section section, string text) and then set it's properties because this method returns Paragraph:

    s.AddParagraph("Page ").SetBold(true).AddPageNumber().AddTextToParagraph(" | ").AddText("BookName").SetBold(false);

  • Call AddParagraph(this Section section, Action<Paragraph> configure) and add content to it using configure action:

    s.AddParagraph(p =>
    {
    p.AddText("Some text");
    p.SetFont(new Font {Name = "Courier", Size = 16f, Color = Color.White});
    p.Alignment = HorizontalAlignment.Center;
    p.BackgroundColor = Color.Blue;
    }


Setting Font of the Paragraph or TextElement is described in Configuring Font article.


Use the AddPageNumber extension method of the Paragraph to print page number. Default numeration is Arabic. Remember to call Section's SetNumerationStyle method if you need to change it to UpperRoman, LowerRoman, UpperLatin, LowerLatin, UpperCyrillic, LowerCyrillic.

You can add URL element to the paragraph using AddUrl method returning FormattedTextElement,

or using AddUrlToParagraph method returning Paragraph.

You can insert inline images into the paragraph by using AddInlineImage.

To get several lines in one paragraph element, use element.AddContent several times.


Please refer various examples below for more information.

        // Create new section:
        Section s = builder.AddDefaultSection();
 
        // Create empty paragraph:
        Paragraph par = s.AddParagraph();
 
        // AddText with font
        par.AddTextToParagraph("Text New 1, ").AddTextToParagraph("Text New 2, ").AddTextToParagraph("Text New 3").SetFont(documentFont);
 
        // AddText with default font:
        s.AddParagraph().AddTextToParagraph("No font provided 1, ").AddTextToParagraph("No font provided 2, ").AddText("No font provided 3");
 
        // Only font size provided:
        s.AddParagraph().AddTextToParagraph("Only font size provided 1, ").AddTextToParagraph("Only font size provided 2, ").SetFontSize(48f);
 
        // Override font size later:
        s.AddParagraph().AddTextToParagraph("Size was 14f, became 48f")
            .SetFont(documentFont)
            .SetFontSize(48f);
 
        // Default font with only font color provided:
        s.AddParagraph().AddTextToParagraph("Only font color provided").SetFontColor(Color.Red);
        s.AddParagraph().AddTextToParagraph("Only font color provided + align center").SetAlignment(HorizontalAlignment.Center).SetFontColor(Color.Green);
 
        // Default font with size and color provided:
        s.AddParagraph().AddTextToParagraph("Size + color provided").SetFontSize(36f).SetFontColor(Color.Green);
 
        // Only font name provided:
        s.AddParagraph().AddTextToParagraph("Only font name provided").SetFontColor(Color.Red);
 
        // Change color, size, name and make it bold:
        s.AddParagraph().AddTextToParagraph("New color, size, name and bold").SetFont(documentFont).SetFontColor(Color.FromRgba(1, 0, 1)).SetFontName(FontNames.Times).SetFontSize(54).SetBold();
 
        // Paragraph SetStrikethrough:
        s.AddParagraph().AddTextToParagraph("Paragraph SetStrikethrough").SetStrikethrough(Stroke.Solid,Color.Red);
 
        // Paragraph Font + SetStrikethrough:
        s.AddParagraph().AddTextToParagraph("Paragraph Font + SetStrikethrough").SetFont(documentFont).SetStrikethrough(Stroke.Solid, Color.Red);

Adding URL: [hide]

    // Underlined and Blue color by default:
    s.AddParagraph().AddUrl("http://www.gehtsoftusa.com/", "GehtSoft LLC.");
 
    // If displayText is not provided, it will be the same as urlValue
    // Customize URL:
    s.AddParagraph().AddUrl("http://www.gehtsoftusa.com/").SetFontColor(Color.Green).SetUnderlineColor(Color.Red);
 
    // Using configurer:
    s.AddParagraph(p =>
    {
    	p.AddContent(() => new List<ParagraphElement>
    	{
             new Url
             {
                    DisplayText = "GehtSoft LLC.",
                    Value = "http://www.gehtsoftusa.com/",
                    Font = new Font
                    {
                        Name = FontNames.Courier,
                        Bold = true,
                        Size = 16,
                        Color = Color.Gray,
                        Underline = Stroke.None
                    }
             }
       });
    });

Adding content in Paragraph configurer [hide]

builder.AddSection(s =>
   {
       s.AddParagraph(p =>
       {
            p.AddText("Some text");
            p.SetFont(new Font {Name = "Courier", Size = 16f, Color = Color.White});
            p.Alignment = HorizontalAlignment.Center;
            p.BackgroundColor = Color.Blue;
            p.SetBorder(b =>
            {
                b.SetWidth(3);
                b.SetStyle(Stroke.Solid);
                b.SetColor(Color.FromRgba(0.6, 0.4, 0.7));
            });
            p.IsOutline = true;
            p.OutlineLevel = 0;
       });
   });
