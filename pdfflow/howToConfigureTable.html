Configuring Table

Brief

The article describes how to add and configure a Table.

Details

Using the library you are able to create tables of different styles: with header or not, simple or with alternating rows colored, with the images in the cells, colspan, rowspan, different borders and alignments, ellipsis in the end of the line, column width in percents, etc...


Creating Table

To add a Table to a Section:

  • Call AddTable method of the TableBuilder class. It adds Table to a Section and helps to set table's properties using tableConfigure Action.

  • Or call AddTable to add an empty Table with default settings and then change settings using extension methods, add TableRows and TableCells.

Then you can set TableName, AltRowStyle and other table properties. Please refer the full list of Table properties.

Methods for building a Table and setting it's properties are described in TableBuilder, e.g. SetName, SetWidth, etc.


Adding content

Where are three ways of adding content to the table:

  • SetDataSource. Please, find "Creating table from datasource" example below.

  • Add TableRow to the table and then add TableCells to the row (described below). Be sure to have the same number of cells in every row. Be careful in calculating the number of columns when setting colspan.

  • Add columns to the table (optionally with header text, width in points, width in percents). And then add rows and cells (described below). There are several methods for adding columns: AddColumn, AddColumn, AddColumn, AddColumn, AddColumn, AddColumn, AddColumnPercent.


Adding Row

There are two methods for adding a TableRow to the table.

  • First AddRow method adds an empty row to the table and returns row.

    You can set row's properties in the same line calling Row's extension methods such as AddTag (adds any object you need), SetHorizontalAlignment, SetVerticalAlignment, SetBorderStyle, SetBackColor, SetStrikethrough, SetFont, SetItalic, etc.

  • The second one AddRow method adds a row and allows to configure it using action.


Adding Cell

Adding TableCells to the row are available with the methods

  • AddCell (adds a cell to the table row and returns row; has action for configuring cell),

  • AddCell (adds a new cell to a row and returns TableCell),

  • AddCellToRow (adds new cell to the row; same as previous AddCell, but returns TableRow),

  • AddCell (adds cell with text content; can set colspan and rowspan; returns TableCell),

  • AddCellToRow (adds new cell with text content; can set colspan and rowspan; same as previous AddCell, but returns TableRow.).

Some of these methods return TableRow, so you can add one more cell to this row in the same line of code. Some of these methods return TableCell, so you can call cell's properties immediately.


Styles and Fonts

You can configure appearence of the content (font, borders, strikethrough...) by setting TableCellStyle:

  • Table's ContentRowStyle (will be applied to all the cells in a table, except of header and footer, they are set with HeaderRowStyle and FooterRowStyle)

  • TableRow's CellsStyle (changes style of single row)

  • TableCell's Style (changes style of single cell)

You can set background color, border, strikethrough, margins and other styles.

Please refer TableCellStyle for the full list of properties.

Among other things, there is TableCellStyle.Font property, allowing you to change Font where you need.

If you need to add text with another font, different from cell.Style.Font, first you have to apply .SetFont(default) to CellsStyle: [show]

If you need to add text with another font, different from cell.Style.Font, first you have to apply .SetFont(default) to CellsStyle: [hide]

	table.AddColumnPercent("Date Received", 10).SetStyle(headerRowStyle)
	.SetFont(default)
	.AddHeaderTextToColumn(" ").AddHeaderText(DateFormat.ToLower()).SetFont(titleFont).SetFontSize(8).SetBold(false).SetItalic();

If you need different fonts in one cell, create Paragraph with several elements: [show]

If you need different fonts in one cell, create Paragraph with several elements: [hide]

        Paragraph par = new Paragraph();
        par.AddText("Times 12 Blue Italic, ").SetFont(Font.Times(12)).SetFontColor(Color.Blue).SetItalic();
        par.AddText((element) =>
        {
            element.SetFont(new Font() { Name = FontNames.Helvetica, Color = Color.Gray }).SetFontSize(8).SetBold(true);
            element.AddContent("Helvetica 8 Gray Bold");
        });
        TableRow row = table.AddRow();
        row.AddCell().SetFont(default).AddParagraph(par);


Events

For your convenience Table has Action<TableCell> CellAdded, Action<TableRow> RowAdded, Action<TableColumn> ColumnCreated, Action<TableCellFormatingArgs> CellFormating and Action<TableRowLayoutEventArgs> RowLayout actions.

Creating table with different cell styles [hide]

        Section _tableSection = builder.AddDefaultSection();
        Table _table = _tableSection.AddTable();
        TableRow _row1 = _table.AddRow();
        // Change cell's Font:
        TableCell _cell11 = _row1.AddCell().SetFont(Font.Times(20));
        _cell11.AddParagraph().AddTextToParagraph("Row 1, Cell 1");
        _row1.AddCell().AddParagraph().AddTextToParagraph("Row 1, Cell 2");
        // Change border style:
        _table.AddRow().SetBorderStyle(Stroke.Dotted).AddCellToRow("Row 2, Cell 1").AddCell().AddParagraph().AddTextToParagraph("Row 2, Cell 2");
        Paragraph _cellPar = new Paragraph();
        _cellPar.AddTextToParagraph("Row 3, Cell 1");
        TableRow _newRow = _table.AddRow();
        // Change Paragraph's Font:
        _newRow.AddCell().SetFont(default).AddParagraph(_cellPar).SetFontColor(Color.Green).SetBold();
        Paragraph _cellPar32 = new Paragraph().AddTextToParagraph("Row 3, Cell 2");
        var style = _table.ContentRowStyle.Clone();
        style.HorizontalAlignment = HorizontalAlignment.Right;
        // Add paragraph and return cell, right aligned:
        TableCell _cell32 = _newRow.AddCell().AddParagraphToCell(_cellPar32).SetStyle(style);
        TableRow _row4 = _table.AddRow();
        // Add strikethrough Paragraph with to cell and return paragraph:
        _row4.AddCell().SetFont(default).AddTag("id").AddParagraph("Row4, Cell 1").SetStrikethrough(Stroke.Solid, Color.Blue);
        // Add Paragraph to cell and return cell, right aligned, red color:
        style.SetBackColor(Color.Red);
        _row4.AddCell().AddTag("id").AddParagraphToCell("Row 4, Cell 2").SetStyle(style);
        // Different Fonts in one cell:
        Paragraph _cellPar51 = new Paragraph();
        _cellPar51.AddText("Row 5, ").SetFont(Font.Times(12)).SetFontColor(Color.Blue).SetItalic();
        _cellPar51.AddText((element) =>
        {
            element.SetFont(new Font() { Name = FontNames.Helvetica, Color = Color.Gray }).SetFontSize(8).SetBold(true);
            element.AddContent("Cell 1");
        });
        TableRow _row5 = _table.AddRow();
        _row5.AddCell().SetFont(default).AddParagraph(_cellPar51);
        _row5.AddCell("Row 5, Cell 2");

Creating table from datasource [show]

Creating table from datasource [hide]

    //Columns of the table:
    [DataContract]
    public class Customer
    {
        [DataMember]
        public string Id { get; set; }
        [DataMember]
        public string CompanyName { get; set; }
        [DataMember]
        public string ContactName { get; set; }
        [DataMember]
        public string ContactTitle { get; set; }
        [DataMember]
        public string Address { get; set; }
        [DataMember]
        public string City { get; set; }
        [DataMember]
        public string PostalCode { get; set; }
        [DataMember]
        public string Country { get; set; }
        [DataMember]
        public string Phone { get; set; }
        [DataMember]
        public string Fax { get; set; }

        public override string ToString()
        {
            return string.Concat($"{Id}\n{CompanyName}\n{ContactName}\n{ContactTitle}\n{Address}\n{City}\n{PostalCode}\n{Country}\n{Phone}\n{Fax}");
        }
    }

    //Fill datasource from json:
    List<Customer> Customers { get; }
    string CustomersJsonFile = Path.Combine(ProjectDir, "Content","customers.json")
    string CustomersJsonContent = File.ReadAllText(CustomersJsonFile);
    Customers = JsonConvert.DeserializeObject<List<Customer>>(CustomersJsonContent);

    //Add table from datasource with alternating row style
    builder.AddDefaultSection().AddTable(table =>
    {
        table.SetName("SimpleTable").SetDataSource(Customers);
        //Set alternating row style:
        table.AltRowStyle = Table.GetDefaultAltRowCellStyle();
    });
