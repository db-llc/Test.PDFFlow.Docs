Creating Line

Brief

The article describes how to create a Line.

Details

Library allows you to add a Line and set it's Width, Style, Color and Length.

Line extension methods are described in LineBuilder.

Method AddLine creates a new Line and returns it.

Then you can set line's properties using SetWidth, SetLength, SetColor, SetStyle (None, Solid, Dashed, Dotted, Double)

ContentItem extension methods from ContentItemBuilder, such as SetMargins, are also available for the line.
