Namespace Gehtsoft.PDFFlow.Builder

Brief

The namespace consists of builder classes to add and configure different content (sections, paragraph, images...)

Details

Classes

BordersBuilder

Class for setting up border's properties (width, style, color...)

ContentItemBuilder

Lets you to set ContentItem properties (margins, border, etc...)

DocumentBuilder

Class lets you to create a new PDF document, add and configure sections and build a document

FormattedTextElementBuilder

Lets you to set FormattedTextElement properties (font, background...) and add content

ImageBuilder

Lets you to add images and inline images and set image properties

LayoutAreaBuilder

Lets you to add repeating areas (headers and footers) to the section

LineBuilder

Allows to create a line and set it's properties

ParagraphBuilder

Lets you to configure Paragraph properties and add content

PredefinedSizeBuilder

Contains functions for converting paper size and orientation

SectionBuilder

Helps you to easy add and configure sections and content

StackFlowBuilder

Lets you to add and configure StackFlow (a range of content items)

TableBuilder

Lets you add and configure a table and it's content

Enumerations

PaperSize

Enumaration of paper sizes (A0...A5)
