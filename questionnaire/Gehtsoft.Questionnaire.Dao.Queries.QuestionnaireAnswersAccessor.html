Brief

The questionnaire answers entity DAO accessor.

Details

The accessor is a class that provides basic CRUD functionality for an entity. The class is based on EF accessor EntityGenericAccessor

In order to support questionnaire-specific behavior this implementation redefines many methods and disables getting next and previous entity.

If base method cannot be used, the warning issued via an obsolete attribute will be show during the compilation and the appropriate exception will be thrown in case method is used.

In most cases it should be enough to use only accessor, however you may need to use directly QuestionnaireAnswersCountSelectQuery , QuestionnaireAnswersSelectQuery and QuestionnaireAnswersSelectQueryBase in order to define more complex CRUD logic.
