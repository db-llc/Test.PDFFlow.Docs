Namespace Gehtsoft.Questionnaire.Dao.Entities

The namespace consists of RDBMS entity definition.

The namespace is defined in package Gehtsoft.Questionnaire.

Pay attention to two entities - QuestionnaireAnswersNumericKeys and QuestionnaireAnswersStringKeys. These tables provides separate storage for the values intended to be searched or used to sort.

The extended update queries will populate search keys into these tables.

The extended search queries will let refer these values by their semantics in select queries. For each such reference another one-to-one join will be established in the query to pick up a key value.
